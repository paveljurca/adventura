# game

* graphics mode (GUI) `java -jar game.jar`
* text mode (CLI) `java -jar game.jar text`

### game commands

```
  batoh  /your backpack
    jdi  /go <room>
    jez  /eat <what>
kdejsem  /current room
  konec  /exit game
   mluv  /talk <to sb> [about sth]
napovez  /help
odemkni  /unlock <room>
    pij  /drink <what>
  poloz  /put <what> [where/who]
  popis  /get detail <what/who>
   save  /save game
  seber  /grab <what> [where/who]
 zamkni  /lock <room>
 ```
